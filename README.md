## Features

- User Authentication (JWT)
- Post Image Upload
- Like & Comment System
- Follow & Unfollow Users
- City-based Feed Filter
- Mood Tagging
- Profile System

## Tech Stack

- Django
- Django REST Framework
- JWT Authentication
- SQLite / PostgreSQL

## Setup Instructions

### Install dependencies
```bash
pip install -r requirements.txt

### Run migrations

python manage.py migrate

### Start server

python manage.py runserver
